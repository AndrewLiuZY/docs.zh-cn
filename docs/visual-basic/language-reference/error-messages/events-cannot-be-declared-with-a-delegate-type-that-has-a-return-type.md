# <a name="events-cannot-be-declared-with-a-delegate-type-that-has-a-return-type"></a>事件不能用具有返回类型的委托类型声明
为函数过程指定了委托。  
  
 **错误 ID：** BC31084  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 指定过程的委托 `Sub` 。
