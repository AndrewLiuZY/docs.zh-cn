# <a name="too-many-files"></a>文件太多
在根目录中创建的文件数超过了操作系统允许的数目，或者打开的文件数超过了配置中的**files =** 设置中指定的数目。SYS 文件。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 如果程序正在打开、关闭或保存根目录中的文件，请将程序更改为使用子目录。  
  
2. 增加你在配置中指定的文件数 **=** 设置。SYS 文件，然后重新启动计算机。
