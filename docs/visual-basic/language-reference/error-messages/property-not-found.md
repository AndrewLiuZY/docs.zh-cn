# <a name="property-not-found"></a>找不到属性
此对象不支持指定的属性。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 检查属性名称的拼写。  
  
2. 检查对象的文档，以确保当对象实际支持 "caption" 或具有类似名称的属性时，不尝试访问类似于 "text" 属性的内容。
