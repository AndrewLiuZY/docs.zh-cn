# <a name="procedure-call-or-argument-is-not-valid-visual-basic"></a>过程调用或参数无效 (Visual Basic)
无法完成调用的某些部分。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 检查允许的参数范围以确保没有任何排列超出允许的值。
