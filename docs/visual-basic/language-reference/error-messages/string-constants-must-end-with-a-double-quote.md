# <a name="string-constants-must-end-with-a-double-quote"></a>字符串常量必须以双引号结束
字符串常量必须以引号开头和结尾。  
  
 **ErrorID：** BC30648  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 请确保字符串文本以引号（"）结尾。 如果从其他文本编辑器中粘贴值，请确保粘贴的字符是有效的引号，而不是类似于它的字符之一，例如 "智能" 或 "大" 引号（"or"）或两个单引号（""）。
