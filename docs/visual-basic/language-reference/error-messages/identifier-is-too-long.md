# <a name="identifier-is-too-long"></a>标识符太长
每个编程元素的名称或标识符限制为1023个字符。 此外，完全限定的名称不能超过1023个字符。 这意味着整个标识符字符串（）的 `<namespace>.<...>.<namespace>.<class>.<element>` 长度不能超过1023个字符，包括成员访问运算符（ `.` ）字符。  
  
 **错误 ID：** BC30033  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 减小标识符的长度。
