# <a name="property-array-index-is-not-valid"></a>属性数组索引无效
提供的值对属性数组索引无效。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 请检查组件的文档，确保索引在指定属性的有效范围内。
