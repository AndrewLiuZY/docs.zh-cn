# <a name="end-raiseevent-must-be-the-first-statement-on-a-line"></a>“End RaiseEvent”必须是某一行中的第一条语句
`End RaiseEvent` 语句跟在冒号 (:) 语句分隔符之后。 `End RaiseEvent` 必须是其源行的唯一语句。  
  
 **错误 ID：** BC31120  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 将多个语句分隔到不同的行。
