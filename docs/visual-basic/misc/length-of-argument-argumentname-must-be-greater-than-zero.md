# <a name="length-of-argument-argumentname-must-be-greater-than-zero"></a>参数“\<argumentname>”的长度必须大于 0
参数的值小于或等于零。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 提供一个大于零的参数。
