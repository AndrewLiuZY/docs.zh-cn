# <a name="structures-cannot-have-inherits-statements"></a>结构不能有“Inherits”语句
与类不同的是，结构不支持继承。  
  
 **错误 ID：** BC30628  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 从结构中删除 `Inherits` 语句。  
  
- 重新设计应用程序以改为使用类。
