# <a name="paramarray-parameter-must-be-an-array"></a>ParamArray 参数必须是一个数组
必须将 `ParamArray` 参数声明为一维数组类型。 声明没有在 `ParamArray` 参数名称后面加上括号。  
  
 **错误 ID：** BC30050  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 在 `ParamArray` 参数名称后面添加一对括号。
