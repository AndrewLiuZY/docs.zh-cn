# <a name="elementname-is-not-a-parameter-of-extension-method-methodname-defined-in-typename"></a>"" \<elementname> 不是 \<methodname> "" 中定义的扩展方法 "" 的 \<typename> 参数
过程调用提供一个不是由该扩展方法定义的附加参数。  
  
 **错误 ID:** BC36585  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 从参数列表中删除多余的参数。
