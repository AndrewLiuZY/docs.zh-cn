# <a name="local-variable-variablename-cannot-be-referred-to-before-it-is-declared"></a>局部变量“\<variablename>”必须先声明，再引用
过程内的变量在声明之前被使用。  
  
 **错误 ID：** BC32000  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 将声明移到变量的第一个引用之前。
