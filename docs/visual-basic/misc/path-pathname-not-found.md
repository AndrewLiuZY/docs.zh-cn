# <a name="path-pathname-not-found"></a>找不到路径“\<pathname>”
操作系统找不到在文件访问或磁盘访问操作期间指定的路径。 文件的路径必须包括查找文件时必须遍历的驱动器规格、目录和子目录。 路径可以是相对路径或绝对路径。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 确保路径包括必需元素。
