# <a name="declare-statements-in-a-structure-cannot-be-declared-specifier"></a>结构中的 "Declare" 语句不能声明为 " \<specifier> "
`Declare` 声明具有在 `Structure` 声明中无效的说明符。  
  
 **错误 ID：** BC30791  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除说明符。
