# <a name="set-is-already-declared"></a>已声明 "Set"
代码块中多次出现 `Set` 语句。 `Set` 语句声明用于为某个属性赋值的 `Set` 属性过程。  
  
 **错误 ID：** BC30444  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除多余的 `Set` 语句。
