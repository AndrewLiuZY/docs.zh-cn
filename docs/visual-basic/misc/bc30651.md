# <a name="property-parameters-cannot-be-declared-byref"></a>\<Property> 参数不能声明为“ByRef”
属性参数必须使用 `ByVal` 修饰符进行声明。  
  
 **错误 ID：** BC30651  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 将属性参数定义为 `ByVal`。
