# <a name="new-is-not-valid-in-this-context"></a>“New”在此上下文中无效
`New` 关键字出现在需要类型名称的位置。 该上下文要求不能与 `New`组合的值类型。  
  
 **错误 ID：** BC30200  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除 `New` 关键字。
