# <a name="division-by-zero-visual-basic-error"></a>除数为零（Visual Basic 错误）
用作除数的表达式的值为零。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 检查表达式中变量的拼写是否正确。 拼写错误的变量可能会隐式创建一个初始化为零的数值变量。  
  
2. 检查之前对表达式中的变量进行的操作，尤其是那些作为参数从其他过程传递给该过程的操作。
