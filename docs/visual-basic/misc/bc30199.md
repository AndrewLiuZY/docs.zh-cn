# <a name="-expected"></a>应输入“(”
过程调用中某个参数列表的开头缺少左括号 (`(`) 字符。  
  
 **错误 ID：** BC30199  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 在该列表的开头添加一个左括号字符。
