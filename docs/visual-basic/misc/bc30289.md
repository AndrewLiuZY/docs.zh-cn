# <a name="statement-cannot-appear-within-a-method-body"></a>语句不能出现在方法体中
过程包含无效语句，如另一个过程声明。 过程声明不能嵌套。  
  
 **错误 ID：** BC30289  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除过程声明，或将其移到封闭过程外部。
