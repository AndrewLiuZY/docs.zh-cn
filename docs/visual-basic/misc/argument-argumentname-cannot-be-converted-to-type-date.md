# <a name="argument-argumentname-cannot-be-converted-to-type-date"></a>参数“\<argumentname>”无法转换为类型“Date”
尝试转换将变量更改为类型 `Date`，但该值未被识别为有效日期。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 查找有意义的转换。
