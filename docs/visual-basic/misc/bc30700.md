# <a name="side-effects-not-valid-during-expression-evaluation-in-this-context"></a>在此上下文中计算表达式期间，副作用无效
在此上下文中，不允许使用更改对象状态的语句。  
  
 **错误 ID：** BC30700  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 更改代码，以便在不更改其他项的状态的情况下计算语句。
