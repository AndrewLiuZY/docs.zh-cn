# <a name="method-methodname-has-no-parameter-named-parametername"></a>方法 " \<methodname> " 没有名为 "" 的参数 \<parametername>
提供的命名参数与目标方法的所有参数均不匹配。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 确保正确提供了命名参数。
