# <a name="character-constant-must-contain-exactly-one-character"></a>字符常量必须正好包含一个字符
字符常量无效，因为它包含多个字符或为 null 字符（所有位均为零）。  
  
 **错误 ID：** BC30004  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 将字符常量更改为单个字符或将 null 字符更改为有效字符。
