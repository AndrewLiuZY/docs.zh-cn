# <a name="with-contexts-and-statements-are-not-valid-in-debug-windows"></a>“With”上下文和语句在调试窗口中无效
`With` 语句只能在源代码中使用。  
  
 **错误 ID：** BC30726  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 从调试代码中删除 `With` 语句。
