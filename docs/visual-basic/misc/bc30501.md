# <a name="shared-cannot-be-combined-with-specifier-on-a-method-declaration"></a>“Shared”不能与方法声明上的“\<specifier>”组合
你试图将 `Shared` 修饰符与方法声明上的此类组合中无效的说明符（例如 `Overridable`、 `NotOverridable`或 `MustOverride`）组合。  
  
 **错误 ID：** BC30501  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除说明符。
