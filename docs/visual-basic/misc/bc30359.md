# <a name="default-can-be-applied-to-only-one-property-name-in-a-type"></a>“Default”只能应用于  中的一个属性名\<type>
尝试在类、接口或结构中对多个属性名应用 `Default` 。  
  
 **错误 ID：** BC30359  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除多余的 `Default` 声明，只保留一个。
