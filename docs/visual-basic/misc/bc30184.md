# <a name="end-enum-must-be-preceded-by-a-matching-enum"></a>“End Enum”前面必须有匹配的“Enum”
出现 `End Enum` 语句而没有相应的 `Enum` 语句。 `End Enum` 前面必须有相应的 `Enum` 语句。  
  
 **错误 ID：** BC30184  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 确保前面的 `Enum` 语句无拼写错误或有效。  
  
2. 确保 `Enum` 成员的格式正确。
