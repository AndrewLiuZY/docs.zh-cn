# <a name="variablename-is-not-a-local-variable-or-parameter-and-so-cannot-be-used-as-a-catch-variable"></a>" \<variablename> " 不是局部变量或参数，因此不能用作 "Catch" 变量
`Try...Catch...Finally` 语句中的变量必须在本地声明，或者是当前过程的参数。  
  
 **错误 ID：** BC31082  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 声明 `Try...Catch...Finally` 语句的本地变量或参数。
