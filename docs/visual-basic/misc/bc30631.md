# <a name="get-statement-must-end-with-a-matching-end-get"></a>“Get”语句必须以匹配的“End Get”结束
`Get` 属性过程必须以 `End Get` 语句结束。  
  
 **错误 ID：** BC30631  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 确保 `Get` 属性过程以 `End Get` 构造结束。
