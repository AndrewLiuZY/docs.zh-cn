# <a name="aggregate-function-name-cannot-be-used-with-a-type-character"></a>聚合函数名称不能与类型字符一起使用
类型字符已包含在聚合函数名称的末尾。  
  
 **错误 ID：** BC36617  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 从聚合函数名称中删除类型字符。
