# <a name="character-character-0xhexadecimal-is-not-allowed-in-xml-literal"></a>XML 文本中不允许出现字符“character”(0x\<hexadecimal>)
在 XML 文本中使用了不受支持的字符。  
  
 **错误 ID：** BC31176  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除不受支持的字符。
