# <a name="end-function-expected"></a>应为“End Function”。
`End` 语句必须与它终止的块匹配。 未正确终止 `Function` 过程定义。  
  
 **错误 ID：** BC30027  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 将 `End Function` 语句添加到过程末尾。
