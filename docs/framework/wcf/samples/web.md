 [基本 HTTP 服务](basic-http-service.md)  
 演示如何使用 WCF Web HTTP 编程模型实现一种基于 HTTP 的基于 RPC 的服务，该一般称为 "POX" （普通旧 XML）服务。
  
 [SOAP 和 HTTP 终结点](soap-and-http-endpoints.md)

 [UriTemplate 表调度程序](uritemplate-table-dispatcher-sample.md)
