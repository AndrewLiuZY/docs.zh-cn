# <a name="igcthreadcontrolsuspensionstarting-method"></a>IGCThreadControl::SuspensionStarting 方法
通知宿主运行时正在为垃圾回收或其他挂起开始线程挂起。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT SuspensionStarting ( );  
```  
  
## <a name="remarks"></a>备注  
 不要在回调过程中重新计划任何线程 `SuspensionStarting` 。
