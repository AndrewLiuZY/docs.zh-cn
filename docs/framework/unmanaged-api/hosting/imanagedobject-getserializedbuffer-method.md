# <a name="imanagedobjectgetserializedbuffer-method"></a>IManagedObject::GetSerializedBuffer 方法
获取此托管对象的字符串表示形式。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetSerializedBuffer (  
    [out] BSTR *pBSTR  
);  
```  
  
## <a name="parameters"></a>参数  
 `pBSTR`  
 弄指向字符串的指针，该字符串是序列化的对象。  
  
## <a name="remarks"></a>备注  
 `GetSerializedBuffer`方法会序列化对象，以便将其封送到客户端。
