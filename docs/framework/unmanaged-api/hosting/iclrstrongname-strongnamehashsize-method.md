# <a name="iclrstrongnamestrongnamehashsize-method"></a>ICLRStrongName::StrongNameHashSize 方法
使用指定的哈希算法获取哈希所需的缓冲区大小。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT StrongNameHashSize (  
    [in]  ULONG   ulHashAlg,  
    [out] DWORD   *pcbSize  
);  
```  
  
## <a name="parameters"></a>参数  
 `ulHashAlg`  
 中用于计算缓冲区大小的哈希算法。  
  
 `pcbSize`  
 弄返回的缓冲区大小（以字节为单位）。  
  
## <a name="return-value"></a>返回值  
 `S_OK`如果该方法已成功完成，则为;否则，表示失败的 HRESULT 值（请参阅列表的[常见 HRESULT 值](/windows/win32/seccrypto/common-hresult-values)）。
