# <a name="igchostcontrol-interface"></a>IGCHostControl 接口
提供一个方法，该方法允许垃圾回收器请求主机更改虚拟内存的限制。  
  
## <a name="methods"></a>方法  
  
|方法|说明|  
|------------|-----------------|  
|[RequestVirtualMemLimit 方法](igchostcontrol-requestvirtualmemlimit-method.md)|请求主机更改虚拟内存的限制。|
