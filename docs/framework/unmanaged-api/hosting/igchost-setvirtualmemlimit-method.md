# <a name="igchostsetvirtualmemlimit-method"></a>IGCHost::SetVirtualMemLimit 方法
设置运行时的虚拟内存的最大大小。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT SetVirtualMemLimit (  
    [in] SIZE_T sztMaxVirtualMemMB  
);  
```  
  
## <a name="parameters"></a>参数  
 `sztMaxVirtualMemMB`  
 中运行时虚拟内存的最大大小（以 mb 为单位）。  
  
## <a name="remarks"></a>备注  
 可以动态更改运行时的虚拟内存的最大大小。
