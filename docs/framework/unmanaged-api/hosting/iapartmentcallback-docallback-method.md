# <a name="iapartmentcallbackdocallback-method"></a>IApartmentCallback::DoCallback 方法
在单元内执行指定的函数。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT _stdcall DoCallback(  
    [in] SIZE_T pFunc,  
    [in] SIZE_T pData  
);  
```  
  
## <a name="parameters"></a>参数  
 `pFunc`  
 中指向要在单元中执行的函数的指针。  
  
 `pData`  
 中指向函数的自变量的指针。
