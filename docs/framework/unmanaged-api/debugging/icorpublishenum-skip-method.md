# <a name="icorpublishenumskip-method"></a>ICorPublishEnum::Skip 方法
按指定的项数在枚举中向前移动光标。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT Skip (  
    [in] ULONG   celt  
);  
```  
  
## <a name="parameters"></a>参数  
 `celt`  
 中游标向前移动的项数。
