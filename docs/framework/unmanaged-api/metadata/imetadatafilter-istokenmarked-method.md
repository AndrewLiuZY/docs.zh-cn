# <a name="imetadatafilteristokenmarked-method"></a>IMetaDataFilter::IsTokenMarked 方法
获取一个值，该值指示指定的元数据标记是否已标记为已处理。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT IsTokenMarked (  
    [in]  mdToken  tk,
    [out] BOOL     *pIsMarked  
);  
```  
  
## <a name="parameters"></a>参数  
 `tk`  
 中要检查处理标记的标记。  
  
 `pIsMarked`  
 弄如果已处理，则该值为 `true` `tk` ; 否则为 `false` 。
