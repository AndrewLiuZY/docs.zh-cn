# <a name="iceegenemitstring-method"></a>ICeeGen::EmitString 方法
向基本代码发出指定的字符串。  
  
 此方法已过时，不应使用。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT EmitString (  
    [in]  LPWSTR    lpString,  
    [out] ULONG     *RVA  
);  
```  
  
## <a name="parameters"></a>参数  
 `lpString`  
 中要发出的字符串。  
  
 `RVA`  
 弄发出的字符串的相对虚拟地址。
