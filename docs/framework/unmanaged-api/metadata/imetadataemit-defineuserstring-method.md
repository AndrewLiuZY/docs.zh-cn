# <a name="imetadataemitdefineuserstring-method"></a>IMetaDataEmit::DefineUserString 方法
获取指定文本字符串的元数据标记。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT DefineUserString (
    [in]  LPCWSTR     szString,
    [in]  ULONG       cchString,
    [out] mdString    *pstk
);  
```  
  
## <a name="parameters"></a>参数  
 `szString`  
 中要存储的用户字符串。  
  
 `cchString`  
 中中的宽字符数 `szString` 。  
  
 `pstk`  
 弄分配的字符串标记。
