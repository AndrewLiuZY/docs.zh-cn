# <a name="cvstruct-structure"></a>CVStruct 结构
包含在安装模块或复合图像时所使用的信息。  
  
## <a name="syntax"></a>语法  
  
```cpp  
typedef struct {  
    short Major;  
    short Minor;  
    short Sub;  
    short Build;  
} CVStruct;  
```  
  
## <a name="members"></a>成员  
  
|成员|描述|  
|------------|-----------------|  
|主要|主版本的内部版本号。|  
|Minor|次版本号。|  
|Sub|子生成号。|  
|构建|内部版本号。|
