# <a name="imetadataemitsave-method"></a>IMetaDataEmit::Save 方法
将当前范围中的所有元数据保存到指定地址处的文件。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT Save (
    [in]  LPCWSTR     szFile,
    [in]  DWORD       dwSaveFlags  
);  
```  
  
## <a name="parameters"></a>参数  
 `wzFile`  
 中要保存到的文件的名称。 如果此值为 null，则内存中的副本将保存到上次使用的位置。  
  
 `dwSaveFlags`  
 [in] 保留。 必须为零。
