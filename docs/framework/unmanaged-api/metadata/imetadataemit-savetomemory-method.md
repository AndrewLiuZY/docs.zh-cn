# <a name="imetadataemitsavetomemory-method"></a>IMetaDataEmit::SaveToMemory 方法
将当前范围中的所有元数据保存到指定的内存区域。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT SaveToMemory (
    [out]  void        *pbData,
    [in]   ULONG       cbData
);  
```  
  
## <a name="parameters"></a>参数  
 `pbData`  
 弄开始写入元数据的地址。  
  
 `cbData`  
 中已分配内存的大小（以字节为单位）。
