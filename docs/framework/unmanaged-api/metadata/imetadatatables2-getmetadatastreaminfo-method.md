# <a name="imetadatatables2getmetadatastreaminfo-method"></a>IMetaDataTables2::GetMetaDataStreamInfo 方法
获取指定索引处的元数据流的名称、大小和内容。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetMetaDataStreamInfo (  
   [in]  ULONG        ix,  
   [out] const char   **ppchName,  
   [out] const void   **ppv,  
   [out] ULONG        *pcb  
);  
```  
  
## <a name="parameters"></a>参数  
 `ix`  
 中请求的元数据流的索引。  
  
 `ppchName`  
 弄指向流名称的指针。  
  
 `ppv`  
 弄指向元数据流的指针。  
  
 `pcb`  
 弄的大小（以字节为单位） `ppv` 。
