# <a name="imetadatatablesgetnumtables-method"></a>IMetaDataTables::GetNumTables 方法
获取当前实例的范围中的表数 `IMetaDataTables` 。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetNumTables (  
    [out]  ULONG   *pcTables  
);  
```  
  
## <a name="parameters"></a>参数  
 `pcTables`  
 弄一个指针，指向当前实例范围中的表数。
