# <a name="isymunmanageddisposedestroy-method"></a>ISymUnmanagedDispose::Destroy 方法
使基础对象释放所有内部引用，并在所有后续方法调用中返回失败。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT Destroy();  
```  
  
## <a name="return-value"></a>返回值  
 如果该方法成功，则 S_OK;否则，E_FAIL 或其他一些错误代码。  
  
## <a name="requirements"></a>要求  
 **标头：** CorSym，CorSym
