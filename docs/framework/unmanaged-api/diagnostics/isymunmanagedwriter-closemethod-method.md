# <a name="isymunmanagedwriterclosemethod-method"></a>ISymUnmanagedWriter::CloseMethod 方法
关闭当前方法。 方法关闭后，不能在其中定义更多符号。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT CloseMethod();  
```  
  
## <a name="return-value"></a>返回值  
 如果该方法成功，则 S_OK;否则，E_FAIL 或其他一些错误代码。  
  
## <a name="requirements"></a>要求  
 **标头：** CorSym，CorSym
